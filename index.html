"Notes"

Sept. 2017 - Dec. 2017
• Audio Engine Programmer, Gameplay Programmer
• Console Rhythm Game
• Team of 2
• Custom C Engine
• Project at DigiPen Institute of Technology

      Yes, you read that right; a "Console Rhythm Game!" In one of the first semester classes at DigiPen, GAM 100, the final team assignment is to create a game with nothing but C code and the console primarily using printf and scanf. The expectation from the class instructors is that students will create something along the lines of a text adventure, but many get creative!


      "Notes" was a very huge learning experience for my teammate and I to create from scratch. The highlights of what I learned while creating this project include:

• Integrating low-level FMOD into a project to stream audio from a file.
• Creating the basic structure and game loop of a beatmania-like rhythm game.
• Creating a music/note track file that is readable by the game.
• Controlling the framerate based on CPU ticks to keep notes from scrolling fast/slow on different computers.
